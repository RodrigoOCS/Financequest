    Olá!

    Existe um nicho de pessoas que (como eu) são interessadas em modelos estatísticos e de aprendizado de máquina e são fascinadas na ideia de modelar o imprevisível mundo das finanças.

    Porém o processo de encontrar uma base de dados, baixá-la, importá-la para um programa e fazer a correta modelagem toma um certo tempo, especialmente quando se está aprendendo modelos e qualquer mínimo gatilho se converte em um brainstorm.

    Para isso, desenvolvi o financequest, para ser possível testar rapidamente um ou outro modelo.

    Além disso, para os devs interessados em python, o site já permite importar o código que executa o modelo, para que seja possível ver um exemplo da aplicação (ou também fazer o famoso CONTROL+C no código do amiguinho).

Link: https://financequest-rocs.herokuapp.com/
